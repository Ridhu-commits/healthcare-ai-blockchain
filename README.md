# AI Applications in Blockchain

This repository contains implementations of **Artificial Intelligence applications applied to Blockchain technology** as part of academic coursework.

## Objective
- To study how AI enhances blockchain systems
- To apply machine learning techniques to blockchain data
- To detect fraud and anomalies in blockchain transactions
- To understand real-world blockchain intelligence use cases

## Scope
This project focuses on analyzing blockchain transaction data using AI models to identify suspicious or fraudulent behavior.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Blockchain transaction datasets (Ethereum / Bitcoin)
- Git & GitHub
